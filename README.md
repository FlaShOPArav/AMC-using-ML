# Automatic Modulation Classification using Machine Learning

## Project Overview
This project implements Automatic Modulation Classification (AMC) using a Convolutional Neural Network (CNN).
The system classifies:
- AM
- FM
- FSK
- BPSK
- QPSK
- QAM

## Tools & Technologies
- Python
- TensorFlow / Keras
- NumPy
- Google Colab
- GitHub

## Methodology
1. Generate synthetic modulated signals
2. Prepare dataset
3. Train CNN model
4. Predict modulation type

## Platform
Google Colab (Cloud-based ML environment)
